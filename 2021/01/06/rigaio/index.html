Rig Tool AIO

Rig Tool AIO is a blender plugin to help rigger. This plugin is still early development and kind of an accident. I’m in the middle of learning rigging character and at that moment, I think most of the step is can be minimize by creating some script.

Feature :

  • Humanoid Rig
  • Vertex Weight Helper

Roadmap :

  • Quadruped Rig
  • Additional Limb
  • Face Rig

Until I post this, Rig Tool AIO still 4 days development, so it still have long way to publish.
