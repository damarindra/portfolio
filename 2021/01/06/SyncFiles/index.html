SyncFiles

Tools : Sync Files
Status : Complete
Engine : Unity 3D

Description

SyncFiles is a tool that I created when developing a game in the RUS Studio. It quite difficult to track the newest assets files and import into the project. SyncFiles will automatically gather all the newest files and replace the old one (if present) into the project directory.

This tool helps you to import all assets by specifying the source folder and the file type, and by clicking the Sync Files, it will automatically synchronize all of the source folder files into project folders. This tools use Update method, so the newest files always win, wether in project folder or source folder. Currently only support on Windows, other platform should be easy to implement it.
