Description

Proformer is my Final Project at College. Proformer is a plugin that help you to create platformer game (sidescroller). Proformer inspired from Elsword, you can create game side scrolling with complex direction, not only using X and Y coordinate, but also Z coordinate, but still using side scrolling aspect.
